# JavaScriptPractice

A structured collection of JavaScript programs organized by difficulty. Each solution includes a problem statement, approach, and time/space complexity analysis.

## Folder Structure

```
JavaScriptPractice/
├── level-1/        # Beginner programs — strings, arrays, objects, DOM basics
├── level-2/        # Intermediate programs — algorithms, logic, string manipulation
└── level-3/        # Advanced programs — coming soon
```

## Level 1

| File | Description |
|------|-------------|
| `absoluteDiff.js` | Find absolute difference between 42 and a number |
| `carObject.js` | Return a car object with predefined properties |
| `changeBgColor.js` | Change background color of a DOM element |
| `check-42.js` | Check if either number or their sum equals 42 |
| `checkCheckbox.js` | Programmatically check a checkbox element |
| `conditionalMath.js` | Multiply two numbers if equal, otherwise add them |
| `countOdds.js` | Count odd numbers in an array |
| `evenIndexChars.js` | Extract characters at even indexes from a string |
| `foursVsZero.js` | Check if count of 4s is greater than count of 0s |
| `fullName.js` | Combine first and last name inputs into full name |
| `helloWorld.js` | Classic hello world program |
| `jsonCreator.js` | Create a JSON string from a key-value pair |
| `longerProperty.js` | Return the longer string between two object properties |
| `middleElements.js` | Extract middle element from two arrays |
| `multiplesOf7.js` | Collect all multiples of 7 between 0 and 99 |
| `passwordValidator.js` | Check if passwords match and meet length requirement |
| `printHeadline.js` | Dynamically create and display an h1 element |
| `reverseArray.js` | Reverse the order of elements in an array |
| `stringReplace.js` | Replace all occurrences of a word in a string |
| `urlConverter.js` | Convert HTTP URL to HTTPS |

## Level 2
